FAQ

Whether the purchase is indefinite or has a validity period.

The purchase is for an unlimited period of time and has no expiry date.

(For Android devices) I cannot find the file after saving it.

The newly saved file can be found in the download folder.

(For IOS devices) I cannot find the saved files folder after updating the application.

Sometimes the new updated application references a different folder (IOS bug). Try this: Press "Browse" then "On My Iphone". You should see the previously created folder here. If it does not work, please contact us.

The message "Program not found" appears.

You are probably using the wrong settings - for example: if you want to create a program for a Haas machine, select the Haas option in the settings panel. If it does not work, please contact us.

I found an error, what should I do?

If you found any bug, please contact us.

I cannot save the file.

On your device, go to settings, find “apps”, then find Macro Mill Plus CNC, select "allowed storage". If it has not worked, please contact us.

After pressing the "Exec" button (the light gray triangle in the upper left corner of the upper bar) the application freezes.

Probably the program you are trying to execute loops endlessly. If you can't find the wrong part in your code, go to the "exec line" and press "Play" or "Play fast". After a while you should recognize what part of the code is repeating. If You are still not sure, press “Stop” and then manually step through the lines using the button “Next”.
