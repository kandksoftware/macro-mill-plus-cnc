Compatibility

Controls compatibility

Macro Mill Plus CNC is compatible with CNC controls such us: Fanuc 0(M), Fanuc 6(M), Fanuc 10(M), Fanuc 11(M), Fanuc 15(M), Fanuc 16(M), Fanuc 18(M), Fanuc 20(M), Fanuc 21(M) and higher, Haas and others using CNC macros based on Fanuc Custom Macro B.

File extensions

Macro Mill Plus CNC can save files with extensions such as: txt, cn, nc, ncc and cnc.

Unit conversion mode

Macro Mill Plus CNC allows you to change between the metric and imperial systems.
